🚨 KERN-REGELN (KRITISCH)

❌ NIEMALS

  • box-shadow verwenden
  • linear-gradient oder radial-gradient
  • Eckige Ecken (border-radius: 0)
  • Border auf farbigen Cards
  • Weiße Schrift auf Brand-Farben (außer Charcoal-BG)
  • Brand-Farben in Charts verwenden
  • Ampel-Labels auf farbigen Cards
  • Button Size Medium (nur Large!)

✅ IMMER

  • Text in Charcoal (#333) - außer auf Charcoal-Hintergrund
  • border-radius: 8px für Cards
  • border-radius: 28px für Buttons
  • Farbige Cards ohne Border
  • Weiße Cards mit Border (2px solid gray-medium)
  • Minimum 30px Padding in Cards
  • Chart-Farben für Datenvisualisierung
  • Ampel-Labels nur auf weißem/Sand-Hintergrund

Checkliste vor Fertigstellung

• Font: "Make it sans"
• Text: Charcoal (#333)
• Farbige Cards: border: none
• Weiße Cards: border: 2px solid gray-medium
• ALLE Cards: border-radius: 8px
• ALLE Buttons: border-radius: 28px, height: 48px
• KEINE box-shadow
• KEINE Gradients (linear/radial)
• Charts: chartColors (NICHT Brand-Farben)
• Ampel-Labels (↑/→/↓): nur auf Weiß/Sand
• Ampel-Logik: ↑=Grün, →=Gelb, ↓=Rot
• Farbige Cards: nur weiße Labels
• Cards mit Ampel-Labels: weiß + border
• Minimum 30px Card-Padding
